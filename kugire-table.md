# 暫定的な K1, K2, K0 一覧表

いま実際に出力されている `kokin-kugire.json` を見ると、K1 は「けり」「む」「や」などの強い終止的形式、K0 は「に」「て」「の」などの接続的形式、K2 は体言止めや曖昧な境界に集まっており、この三分法はかなりうまく働いています。

| クラス | 基本的な考え方     | 典型的な品詞・活用                                               | 例                                           | 運用方針                         |
| ------ | ------------------ | ---------------------------------------------------------------- | -------------------------------------------- | -------------------------------- |
| K1     | 文法的に強い区切り | 終止形、命令形、終助詞、強い係助詞、終止に準ずる形               | 「けり」「む」「らむ」「や」「か」「みよ」   | 原則として句切れ候補にする       |
| K2     | 保留・中間領域     | 名詞終止、体言止め候補、連語、曖昧な副詞・助詞、即断しにくい形式 | 「白雪」「心」「てふ」「まにまに」「を」など | 自動では保留にし、あとで確認する |
| K0     | 非候補             | 格助詞、接続助詞、用形、連体形、明らかな継続形                   | 「に」「て」「の」「より」「なく」「ふかく」 | 原則として句切れ候補にしない     |

これを、もう少し細かく品詞・形式ごとにばらすと、次のようになります。

## K1 一覧

| 形式                                             | 判定    | 備考                                 |
| ------------------------------------------------ | ------- | ------------------------------------ |
| 活用語の終止形                                   | K1      | 最も基本的な強候補                   |
| 命令形                                           | K1      | 発話の切れが明示的                   |
| 推量・詠嘆などの終体で、句末で明確に言い切るもの | K1      | 「む」「らむ」など                   |
| 終助詞                                           | K1      | 「かな」「かも」などもここへ入れうる |
| 係助詞のうち、切れを強く作るもの                 | K1      | とくに「や」「か」「こそ」は強い     |
| 已然形で句末に立つもの                           | K1 寄り | 係り結びとの関係で強候補             |

実際の出力例では、「けり」「む」「や」「か」などが安定して K1 に入っています。00001 の「けり」「む」、00329 の「や」、00869 の「らむ」などがそれです。

## K2 一覧

| 形式                                               | 判定 | 備考                           |
| -------------------------------------------------- | ---- | ------------------------------ |
| 名詞終止                                           | K2   | 体言止め候補だが即 K1 にしない |
| 体言止めらしく見える句末                           | K2   | 文芸的な揺れを残す             |
| 表面上は切れそうだが、文法だけでは断定しにくいもの | K2   | 中間領域として保持             |
| 連語・慣用的助詞句                                 | K2   | 例: 「てふ」                   |
| 副詞・副詞的表現で句末に立つもの                   | K2   | 例: 「はや」「まにまに」       |
| 品詞解析がやや不安定なもの                         | K2   | 自動判定を保留する             |

実例では、00001 の第三句末「を」が K2、00325 の「白雪」が K2、00723 の「心」が K2、00953 の「まにまに」が K2 に入っています。K2 は、まさに「ここは文法だけでは切れとも非切れとも言い切らない」という領域として働いています。

## K0 一覧

| 形式                     | 判定 | 備考                             |
| ------------------------ | ---- | -------------------------------- |
| 格助詞                   | K0   | 「に」「の」「と」「より」など   |
| 接続助詞                 | K0   | 「て」など                       |
| 用形                     | K0   | 明らかに次へ続く                 |
| 連体形                   | K0   | 後続を要求するので原則非候補     |
| 形容詞・形容動詞の用形   | K0   | 「ふかく」「なく」「をしく」など |
| 途中の修飾関係にあるもの | K0   | 明示的な切れではない             |

実例では、「に」「て」「の」が非常に安定して K0 に入っています。00018 の第一句末「の」、第四句末「て」、00954 の第一句末「の」、第四句末「る」は、いずれもその典型です。

ここで大事なのは、**係助詞を一律に K1 にしない方がよい**という点です。現在の出力では「は」「も」も K1 に入っている例がありますが、これはやや強すぎます。たとえば 00080 の第二句末「も」、00355 の第一句末「も」、00746 の第四句末「も」は K1 になっていますが、運用上は次のように細分してよいと思います。

| 係助詞 | 推奨クラス   | 備考                                   |
| ------ | ------------ | -------------------------------------- |
| や     | K1           | 強い切れ                               |
| か     | K1           | 強い切れ                               |
| こそ   | K1           | 強い切れ                               |
| ぞ     | K1 または K2 | 文脈次第だが強め                       |
| は     | K2           | 提示であって切れとは限らない           |
| も     | K2           | 添加・取り立てであって切れとは限らない |

この修正を入れると、K1 がより「強い文法的区切り」に集中し、K2 が「切れそうだが強くはない」領域としてさらに見やすくなります。

したがって、現段階での運用表を簡潔に書くなら、こうなります。

| クラス | 判定対象                                                         |
| ------ | ---------------------------------------------------------------- |
| K1     | 終止形、命令形、終助詞、「や」「か」「こそ」など強い係助詞       |
| K2     | 名詞終止、体言止め候補、「は」「も」、連語、副詞的句末、判定保留 |
| K0     | 格助詞、接続助詞、用形、連体形、継続形                           |

そして、これを文章で定義すると、次のようになります。

「K1 は、句末に現れたとき文法的に強い切れを形成する形式である。K2 は、体言止めや弱い係助詞など、文法的に切れの可能性はあるが即断を避けるべき保留領域である。K0 は、格助詞・接続助詞・用形など、明らかに次へ続く形式であり、原則として句切れ候補にしない。」
